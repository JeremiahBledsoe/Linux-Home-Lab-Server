# Linux Home Lab Server

Set up a personal home lab with Linux servers to practice system administration, remote access, and network management.

## Tools & Tech
- Linux (Debian)
- RDP/SSH
- Networking & Firewall configurations
- User account management

## Key Tasks
- Set up Linux server with SSH and administrative tools
- Managed users, permissions, and network/firewall settings
- Practiced server administration and troubleshooting

## Skills Demonstrated
- Linux server administration
- Secure remote access setup
- System maintenance and troubleshooting
